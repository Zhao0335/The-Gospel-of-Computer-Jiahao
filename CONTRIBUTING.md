# 🕊️ 贡献指南

凡愿意参与编纂《嘉豪福音》的弟兄姊妹，请遵循以下神圣法则：

1. **不要直接推送到主分支 (Do not push to `main` directly)**
   正如人不能直视神的面孔，你也不可直接修改 `main`。请 Fork 本仓库，创建你自己的分支（如 `feat/book-of-摸鱼`）。
2. **保持经文体裁 (Maintain the Biblical Tone)**
   新增内容请使用“章:节”的格式（如 `X:Y`），并保持《圣经》和合本般的古典、庄重且带有一丝荒诞的幽默感。
3. **Commit Message 规范**
   建议使用祈使句，并在开头加上 Emoji。例如：
   - `✨ feat(经文): 增加《摸鱼传道书》`
   - `🐛 fix(错别字): 修复创世记中的标点遗漏`
4. **提交 PR (Pull Request)**
   提交 PR 即视为你的代码受洗。审查者（Maintainer）会在合并前诵读你的经文。
