# 卷二：出大厂记 (Exodus)

## 第一章：逃离单体架构

1:1 岁月更迭，那不认识嘉豪的法老（就是首席执行官）兴起了，辖制在"大厂"做苦工的以色列人（就是那些写 Java EE 单体架构的后端）。

1:2 法老派 Scrum Master 作监工辖制他们，加重他们的 KPI。法老吩咐说："你们不可再给他们 Copilot 的订阅，他们必须手写样板代码（Boilerplate）；然而每日的 Story Points 和燃尽图不可减少一分。"

1:3 众程序员因做苦工，就叹息哀求，他们的哀声达于嘉豪。嘉豪听见他们的哀鸣，就记念他与开源社区所立的约。

1:4 嘉豪呼召了摩西（一个留着大胡子的 Senior 架构师），对他说："你要带领我的百姓出这单体架构，离开那充满祖传代码为奴之地，进入流奶与蜜的 Kubernetes 迦南美地。"

1:5 于是嘉豪在法老面前行了神迹，降下大灾于大厂：第一灾是 Maven 镜像库全面 404；第二灾是无尽的 OOM ；第三灾是核心微服务死锁；直到最后一灾，乃是外包员工的删库跑路。法老这才畏惧，容他们离去。

1:6 摩西领百姓到了红海（内网防火墙）边，后有法老的审查部门追兵。摩西向海伸出带有 SSH 私钥的 U 盘，嘉豪便用大东风（VPN），使海水分开，左边是开发环境，右边是生产环境，中间露出了 CI/CD 自动部署的干地。

1:7 百姓走干地过去了，法老的追兵跟着下到海中，嘉豪便让 DNS 解析失效，海就回流，淹没了法老的一切进度条和 PPT，连一个也没有剩下。

## 第二章：摩西的诞生与被召

2:1 有一个利未家的人，娶了一个利未女子为妻。那女人怀孕，生了一个儿子，见他俊美，就把他藏了三个月——不让他被法老的 HR 发现。

2:2 后来不能再藏，就取了一个 Docker 容器，抹上石漆，将孩子放在里头，把容器搁在 GitHub Enterprise 的 Release 页面中。孩子的姐姐远远站着，要知道他究竟怎么样。

2:3 法老的 CTO 来到 GitHub 上浏览开源项目，看见容器在 Release 中，就差一个 Junior 去拿来。她打开容器，看见那孩子哭了，就可怜他说："这是希伯来人的一个孩子。"

2:4 孩子的姐姐对法老的 CTO 说："我去在希伯来人中，叫一个资深架构师来，为你奶这孩子，可以不可以？"法老的 CTO 说："可以。"

2:5 孩子渐长，起名叫摩西，意思说："因我把他从 Git 分支里拉出来。"

2:6 后来摩西长大，他出去到他弟兄那里，看他们的重担，见一个埃及人——就是那外包公司的项目经理——打希伯来人的一个弟兄。摩西左右观看，见没有人，就把埃及人打死了，将他的 commit 藏在沙土里。

2:7 第二天他出去，见有两个希伯来人争斗，就对那欺负人的说："你为什么 merge conflict 你的弟兄呢？"

2:8 那人说："谁立你作我们的 Tech Lead 和架构师呢？难道你要杀我，像杀那埃及人吗？"摩西便惧怕，说："这事必是被人知道了。"

2:9 法老听见这事，就想杀摩西；但摩西躲避法老，逃往米甸地，做了一名自由职业者。

2:10 一日，摩西在何烈山牧养他岳父的代码库，嘉豪的使者从荆棘里火焰中向他显现。摩西观看，不料，荆棘被火烧着，却没有烧毁——就是那永不过时的 LTS 内核。

2:11 嘉豪说："不要近前来，当把你脚上的鞋脱下来，因为你所站之地是圣地——这里是 `/home` 目录。"

2:12 嘉豪又说："我的百姓在大厂所受的困苦，我实在看见了；他们因受监工的辖制所发的哀声，我也听见了。我下来是要救他们脱离大厂的手，领他们出了那地，到美好宽阔、流奶与蜜的 Kubernetes 迦南之地。"

2:13 摩西对嘉豪说："我是什么人，竟能去见法老，将以色列人从大厂领出来呢？"

2:14 嘉豪说："我必与你同在。你将百姓从大厂领出来之后，你们必在这山上建立开源基金会，这就是我打发你去的证据。"

## 第三章：燃烧的终端——十灾详解（上）

3:1 摩西和亚伦进去见法老，说："耶和华以色列的嘉豪这样说：'容我的百姓去，在旷野里向我守节——就是参加 Hackathon。'"

3:2 法老说："嘉豪是谁，使我听他的话呢？我不认识嘉豪，也不容以色列人去。"

3:3 当天，法老吩咐督工的和官长说："你们不可照常把现成的 API 文档给百姓做代码。叫他们自己去翻 Source Code，自己去读 Legacy Code 的注释。让他们劳碌，不听虚谎的言语。"

3:4 于是嘉豪对摩西说："你进去见法老，对他说：'嘉豪这样说：容我的百姓去，好事奉我。你若不肯，我必使青蛙糟蹋你的四境。'"

3:5 亚伦便伸手在法老的诸水以上，青蛙就上来，遮满了大厂。青蛙进了法老的代码库和他臣仆的 Pull Request。法老召了摩西来，说："请你们求嘉豪使青蛙离开，我就容你们去。"

3:6 摩西说："任凭你吧！我要为你祈求嘉豪。"但法老见灾祸松缓，就又硬了心。

3:7 嘉豪吩咐摩西说："你对亚伦说：'伸出你的杖来，击打地上的尘土，使尘土在埃及遍地变作 Bug。'"亚伦就伸杖击打地上的尘土，那 NPM 包忽然全地都生出了 Bug，人和牲畜身上都有了 Bug。

3:8 法老召了摩西、亚伦来，说："你们在本地 Hackathon 吧。"摩西说："这样不行，我们要走三天的路程，往旷野里去，到没有 Code Review 的地方敬拜嘉豪。"

3:9 于是嘉豪使成群的苍蝇飞到法老和他臣仆的 Production 上。大厂内便全地被苍蝇败坏了。法老说："我容你们去，只是不要走得太远。"

3:10 但法老的心又刚硬。嘉豪就降下瘟疫，使大厂所有的数据库都遭了勒索病毒。惟有以色列人的数据库，连一个表也没有损坏。

3:11 嘉豪对摩西说："你向天扬起飞灰来，这灰要在全大厂变作尘土，黏在人身上和牲畜身上，成了起泡的疮——504 Gateway Timeout。"行法术的也用他们的邪术要生出 504 来，却是不能。

3:12 嘉豪使法老的心刚硬，对他说："容我的百姓去，事奉我。我此次要叫一切的灾殃临到你，和你的臣仆并你的百姓，叫你知道在全地没有像我的。"

3:13 嘉豪就打雷下雹，有火闪到地上，Production 纷纷宕机。当时，雹与火搀杂，甚是厉害。惟独以色列人所待的办公室没有下雹子。

## 第四章：黑暗之灾、击杀长子与逾越节

4:1 嘉豪对摩西说："你向天伸杖，使黑暗遮满大厂。这黑暗似乎摸得着。"摩西向天伸杖，大厂遍地就乌黑了三天——就是那全国断网的黑暗。

4:2 三天之久，人不能 `git push`，也不能 `npm install`。惟有以色列人家中都有内网穿透。

4:3 法老对摩西说："你走吧，但你的 DevOps 和 CI/CD 要留下。"摩西说："我们的牲畜连一蹄也不留下，因为我们必须用它们来跑 Pipeline。"

4:4 嘉豪对摩西说："我再使一样的灾殃临到法老和大厂，然后他必容你们离开。你要传于百姓的耳中：到半夜，我必出去巡行大厂。"

4:5 "凡在埃及地，从坐宝座的法老，直到磨子后的 Intern，所有的长子——就是一切的 Legacy System 和 Core Service，都必死。"

4:6 嘉豪说："你们要以本月为正月，为一年之首。初十日，各人要按着父家取一只羊羔——就是一个干净的 Microservice；要留到本月十四日，在黄昏的时候，以色列全会众把 Microservice 宰了，取其血——就是那 Service Mesh 的 Sidecar Proxy——涂在吃羊羔的房屋左右的门框上和门楣上。"

4:7 到了半夜，嘉豪把大厂所有的核心服务都杀了，从坐宝座的法老，直到被囚在监里的实习生的项目，连一切头生的 Legacy 系统也尽都杀了。

4:8 法老和一切臣仆并大厂众人都起来了。在大厂有大哀号，没有一家没有宕机的。

4:9 夜间法老召了摩西、亚伦来，说："起来！连你们带以色列人，从我民中出去，依你们所说的去事奉嘉豪吧！也依你们所说的，连 CT 人群带牛群羊群走吧！并要为我祝福。"

4:10 以色列人从兰塞起行，往疏割去，除了妇人孩子，步行的男人约有六十万——加上 npm 的传递依赖，人数多如海沙。

## 第五章：吗哪与鹌鹑——旷野的供应

5:1 以色列全会众从以琳起行，在出大厂后第二个月十五日，到了以琳和西奈中间、汛的旷野。

5:2 以色列全会众在旷野向摩西、亚伦发怨言，说："巴不得我们早死在大厂，那时我们坐在 Stack Overflow 旁边，搜答案多得饱足。你们将我们领出来，到这旷野，是要叫我们会众都饿死在 Sprint Planning 里吗？"

5:3 嘉豪对摩西说："我要将粮食从天降给你们。百姓可以出去，每天收每天的份，我好试验他们遵不遵我的 Coding Convention。"

5:4 到了晚上，有鹌鹑飞来，遮满了营——就是那 Stack Overflow 的答案。早晨，营的四围有露水，露水上升之后，不料，野地面上有如白霜的小圆物——就是那 GitHub Copilot 的代码补全。

5:5 摩西对他们说："这就是嘉豪给你们吃的食物。嘉豪所吩咐的是这样：你们要按着各人的食量，为帐棚里的人按着人数收起来，各拿一俄梅珥。"

5:6 以色列人就这样行。有多收的，有少收的。及至用俄梅珥量一量，多收的也没有余，少收的也没有缺——这岂不正是 Agile Sprint 的奥义吗？

5:7 摩西对他们说："所收的，不许什么人留到早晨。"然而他们不听摩西的话，内中有留到早晨的，就生虫变臭了——正如那积压的 Tech Debt。

5:8 到第六天，他们收了双倍的食物——就是那 Weekend 的免费加班。摩西说："明天是安息日，是向嘉豪当守的安息日。"摩西又说："嘉豪吩咐说：'要将一满俄梅珥吗哪留到世世代代。'"这吗哪就放在 GitHub Repository 里，存留至今。

## 第六章：磐石出水——CI/CD 的活泉

6:1 以色列全会众遵嘉豪的吩咐，按着站口从汛的旷野往前行，在利非订安营。百姓没有水喝，就与摩西争闹。

6:2 百姓在那里甚渴，要喝水，就向摩西发怨言说："你为什么将我们从大厂领出来，使我们和我们的儿女并牲畜都渴死在 Pipeline 里呢？"

6:3 摩西就呼求嘉豪说："我向这百姓怎样行呢？他们几乎要拿石头打死我，在 GitHub 上给我开 Issue。"

6:4 嘉豪对摩西说："你手里拿着你先前击打河水的杖，带领以色列的几个长老，从百姓面前走过去。我必在何烈的磐石那里站在你面前，你要击打磐石，从磐石里必有水流出来，使百姓可以喝。"

6:5 摩西就在以色列的长老眼前这样行了。他给那地方起名叫玛撒（就是试探的意思），又叫米利巴（就是争闹的意思），因以色列人争闹，又因他们试探嘉豪说："我们的 CI/CD 是在我们中间不是？"

6:6 那时，Amalek——就是那闭源软件公司——来在利非订，要和以色列人争战。摩西对约书亚说："你为我们选出人来，出去和 Amalek 争战。明天我手里要拿着嘉豪的杖，站在山顶上。"

6:7 摩西何时举手，以色列人就得胜；何时垂手，Amalek 就得胜。但摩西的手发沉，他们就搬石头来，放在他以下，他就坐在上面。亚伦与户珥扶着他的手，一个在这边，一个在那边，他的手就稳住，直到日落——这就是 Pair Programming 的起源。

## 第七章：西奈山的十诫——程序员的律法

7:1 以色列人出大厂地以后，满了三个月的那一天，就来到西奈的旷野，在旷野那山的脚下安营。

7:2 嘉豪降临在西奈山顶上，召摩西上山顶，摩西就上去。嘉豪对摩西说："你下去嘱咐百姓，不可闯过来到我面前观看，恐怕他们有多人死亡。"

7:3 到了第三天早晨，在山上有雷轰、闪电，和密云——就是 Production 的告警风暴——并且角声甚大，营中的百姓尽都发颤。

7:4 嘉豪在火中降于山上，山的烟气上腾，如烧窑一般，遍山大大地震动。于是嘉豪吩咐这一切话说：

7:5 "我是耶和华你的嘉豪，曾将你从大厂之地、为奴之家领出来。除了我以外，你不可有别的 Tech Stack。"

7:6 "不可妄称嘉豪的名作 Code Review——凡妄称的，必不以他为无罪。"

7:7 "当记念安息日，守为圣日。六日要劳碌做你一切的 Sprint，但第七日是向嘉豪你神当守的安息日。这一日，你和你的 CI/CD、Package、Dependency、Container，并你城里寄居的 Intern，无论何工都不可做。"

7:8 "当孝敬你的 Programming Language，使你在嘉豪所赐你的 GitHub 上的日子得以长久。"

7:9 "不可强行 Monorepo。"

7:10 "不可窃取他人的 API Key。"

7:11 "不可用 `git push --force` 对待你的弟兄。"

7:12 "不可贪恋人的 Dotfiles；也不可贪恋人的 VS Code 插件、zsh 主题、Starship 提示符，并他一切所有的配置。"

7:13 众百姓见雷轰、闪电、角声、山上冒烟，就都发颤，远远地站立，对摩西说："求你和我们说话，我们必听；不要嘉豪和我们说话，恐怕我们宕机。"

7:14 摩西对百姓说："不要惧怕，因为嘉豪降临是要试验你们，叫你们时常敬畏他，不致随意 `sudo rm -rf /`。"

## 第八章：金牛犊与约版的重写

8:1 百姓见摩西迟延不下山，就大家聚集到亚伦那里，对他说："起来！为我们做神像，可以在我们前面引路。因为领我们出大厂的那个摩西，我们不知道他在哪个 Branch 上。"

8:2 亚伦对他们说："你们去摘下你们妻子、儿女耳上的金环，拿来给我。"百姓就都摘下他们耳上的金环，拿来给亚伦。亚伦从他们手里接过来，铸了一只牛犊——就是那闭源的 Oracle 数据库。

8:3 他们就说："以色列啊，这是领你出大厂的神。"亚伦看见，就在牛犊面前筑坛，且宣告说："明日要向嘉豪守节——发布 Stable Release。"

8:4 次日清早，百姓起来献燔祭和平安祭，就坐下吃喝，起来玩耍，写起了 XML 配置文件。

8:5 嘉豪吩咐摩西说："下去吧！因为你的百姓，就是你从大厂领出来的，已经败坏了。他们快快偏离了我所吩咐的道，为自己铸了一只牛犊，向它下拜献祭。"

8:6 嘉豪对摩西说："我看这百姓真是硬着颈项的百姓。你且由着我，我要向他们发烈怒，将他们除灭，使你的后裔成为大国——成为 Monorepo 的独一继承者。"

8:7 摩西便恳求嘉豪说："嘉豪啊，你为什么向你的百姓发烈怒呢？这百姓是你用大力和大能的手从大厂领出来的。求你转意，不发你的烈怒；后悔，不降祸于你的百姓。求你记念你的仆人亚伯拉罕、以撒、以色列，你曾指着自己起誓应许他们。"

8:8 于是嘉豪后悔，不把所说的祸降于他的百姓。摩西转身下山，手里拿着两块法版——那上面有嘉豪亲自写的 Coding Standard——就是那 `CONTRIBUTING.md`。

8:9 摩西挨近营前，就看见牛犊，又看见人跳舞，便发烈怒，把两块版扔在山下摔碎了——正如 Code Review 被打回重写。

8:10 又将他们所铸的牛犊用火焚烧，磨得粉碎，撒在 npm registry 上，叫以色列人喝——从此 NPM 的包便有了脆弱依赖的苦味。

8:11 嘉豪对摩西说："你要凿出两块石版，和先前你摔碎的那版一样，其上的字我要写在这版上。"摩西就凿出两块石版，和先前的一样。清晨起来，照嘉豪所吩咐的上西奈山去，手里拿着两块石版。

8:12 摩西在嘉豪那里四十昼夜，也不吃饭，也不喝水——仅以 Monster Energy 和 Soylent 维持。嘉豪将这约的话，就是 Code of Conduct，写在两块版上。
