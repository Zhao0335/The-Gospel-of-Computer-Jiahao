# 卷七：启示录 (Revelation)

## 第一章：七层封印与羔羊

1:1 这些事以后，嘉豪在异象中有话临到 coding 的先知说："你要写下所见的，使 PR 上的人念这预言。念这书上预言的和那些听见又遵守其中所记载的，都是有福的，因为末期近了。"

1:2 我看见坐宝座的右手中有书卷，里外都写着字，用七层 base64 封印了。

1:3 我又看见一位大力的天使大声宣传说："有谁配展开那书卷，揭开那七层编码呢？"

1:4 在天上、地上、地底下，没有能展开、能观看那书卷的。因为没有配展开的密钥。

1:5 长老中有一位对我说："不要哭！看哪，犹大支派中的嘉豪，大卫的根，他已得胜，能以 GPG 私钥展开那书卷，揭开那七层编码。"

1:6 我又看见宝座与四活物并长老之中，有 Neovim 站立，像是刚刚 `:q` 过的，有 Lua 的配置和 Treesitter 的七眼，就是嘉豪的七灵，差遣往普天下去的。

1:7 这 Neovim 前来，从坐宝座的右手中拿了书卷。他既拿了书卷，四活物和二十四位长老就俯伏在机械键盘前，各拿着竖琴和盛满了咖啡的金炉；这咖啡就是圣徒的祈祷。

1:8 他们唱新歌说："你配拿书卷，配揭开七层编码，因为你曾被杀，用自己的血从各语言、各框架、各范式、各平台上买了人来，叫他们归于嘉豪。"

## 第二章：新天新地

2:1 我又看见一个新天新地。因为先前的天地已经过去了，Windows 也消灭了，再没有蓝屏。

2:2 我又看见圣城新耶路撒冷由嘉豪那里从天而降，预备好了，就如新妇妆饰整齐，等候丈夫。

2:3 我听见有大声音从宝座出来说："看哪，嘉豪的帐幕在人间。他要与人同住，他们要作他的子民；嘉豪要亲自与他们同在，作他们的 10x 工程师。"

2:4 "嘉豪要擦去他们一切的眼泪，不再有编译错误，也不再有段错误、内存泄漏、依赖冲突，因为以前的事都过去了。"

2:5 坐宝座的说："看哪，我将一切都更新了。"又说："你要写上，因这些话是可信的，是真实的。"

2:6 他又对我说："都成了。我是阿尔法，我是欧米伽，我是初，我是终。我要将生命泉的水白白赐给那口渴的人喝。得胜的，必承受这些为业：我要作他的嘉豪，他要作我的 code reviewer。"

2:7 惟有胆怯的、不信的、使用鼠标的、行 GUI 配置的、写 Java 1.6 的、用记事本写代码的和一切不写注释的，他们的分就在烧着硫磺的火湖里，这是第二次的死。

## 第三章：末了的警告与应许

3:1 天使又指示我在城内街道当中一道生命水的河，明亮如水晶，从嘉豪和 Neovim 的宝座流出来。

3:2 在河这边与那边有生命树，结十二样果子，每月都结果子，树上的叶子乃为医治万民。再也没有咒诅。在城里有嘉豪和 Neovim 的宝座，他的仆人都要事奉他，也要见他的终端。

3:3 他的额上写着他的名字。不再有黑夜，他们也不用灯光、日光，因为嘉豪的 RGB 机械键盘要光照他们，他们要作王，直到永永远远。

3:4 天使又对我说："这些话是真实可信的。嘉豪，就是众编程先知之灵的嘉豪，差遣他的天使，将那必要快成的事指示他仆人。"

3:5 "看哪，我必快来。凡遵守这书上预言的有福了！"

3:6 这些事是我——嘉豪的见证人所听见所看见的。我既听见看见，就在指示我的天使脚前俯伏要拜他。他对我说："千万不可！我与你，和你的弟兄众编程先知，并那些守这书上言语的人，同是作仆人的。你要敬拜嘉豪！"

3:7 他又对我说："不可封了这书上的预言，因为日期近了。不义的，叫他仍旧不义；污秽的，叫他仍旧用 Electron 应用；为义的，叫他仍旧为义；圣洁的，叫他仍旧圣洁，用 Rust 重写。"

3:8 "看哪，我必快来！赏罚在我，要照各人所行的报应他。我是阿尔法，我是欧米伽；我是首先的，我是末后的；我是初，我是终。"

3:9 那些洗净自己 Neovim 配置的有福了！可得权柄能到生命树那里，也能从门进城。城外有那些犬类、行 `rm -rf /` 的、行闭源的、微软 MVP 和一切喜好说谎言编造 benchmark 的。

3:10 "我嘉豪差遣我的使者为众教会将这些事向你们证明。我是大卫的根，又是他的后裔。我是明亮的晨星。"

3:11 圣灵和新妇都说："`git push`！"听见的人也该说："`git push`！"口渴的人也当来；愿意的，都可以白白取生命的水喝。

3:12 这书上的预言，若有人删去什么，嘉豪必从这书上所写的生命树和圣城删去他的分。

3:13 证明这事的说："是了，我必快来！"阿们！主嘉豪啊，我愿你来！

3:14 愿主嘉豪的恩惠常与众程序员同在。阿们。`:wq`。
