# 卷四：诗篇 (Psalms)

## 第一篇：不从恶人的计谋

1:1 不从 Scrum Master 的计谋，不站 Product Manager 的道路，不坐 JIRA 的座位，这人便为有福。

1:2 惟喜爱嘉豪的 Coding Standard，昼夜思考 Clean Code，这人便为有福。

1:3 他要像一棵树栽在 GitHub 溪水旁，按时结出 Merge Request，叶子也不枯干。凡他所 Commit 的尽都顺利。

1:4 恶人并不是这样，乃像 CI Pipeline 失败的 Build Log，被风吹散。

1:5 因此当 Code Review 的时候，恶人必站立不住；罪人在义人的 Repository 中也是如此。

1:6 因为嘉豪知道义人的 Branch，恶人的 Branch 却必 `git reset --hard`。

## 第八篇：嘉豪的名在全地何其美

8:1 嘉豪我们的主啊，你的名在全 GitHub 何其美！你将对 Neovim 的荣耀彰显于天。

8:2 你因 Junior Developer 的缘故，从 Stack Overflow 和 Copilot 的口中建立了能力，使仇敌和报仇的——就是那闭源公司——闭口无言。

8:3 我观看你指头所造的终端，并你所陈设的 Moonlander 分体键盘。

8:4 便说，人算什么，你竟顾念他？凡人算什么，你竟眷顾他，还赐他 LSP 自动补全？

8:5 你叫他比天使微小一点，并赐他荣耀尊贵为冠冕——就是那 Senior 的 Title。

8:6,7,8 你派他管理你手所造的，使万物，就是一切的 Protocol、API、Package、Dependency，都服在他的 Makefile 下。

8:9 嘉豪我们的主啊，你的名在全 GitHub 何其美！

## 第十九篇：诸天述说嘉豪的荣耀

19:1 诸天述说嘉豪的荣耀，穹苍传扬他 `~/.config` 的手段。

19:2 这日到那日发出 Commit Message，这夜到那夜传出 200 OK。

19:3 无言无语，也无 Status Code 可听。他的 Pipeline 通遍天下，他的 CI Log 传到地极。

19:4 嘉豪在其间为太阳安设帐幕——就是那永不停机的 Kubernetes Cluster。

19:5 太阳如同新 Deployment 出 Green-Blue Deploy，又如 `git push` 的勇士欢然奔路。

19:6 它从天这边出来，Health Check 到天那边，没有一物被隐藏不得它的日志。

19:7 嘉豪的律法全备，能苏醒人心——如同 `strace` 追踪每一个 syscall；嘉豪的法度确定，能使 Junior 有智慧——如同 `gdb` 定位每一个 segfault。

19:8 嘉豪的训词正直，能快活人的心；嘉豪的命令清洁，能明亮人的 `tmux` 窗口。

19:9 嘉豪的道理洁净，存到永远——不像那过时的 Framework；嘉豪的典章真实，全然公义。

19:10 都比金子可羡慕，且比极多的 Stack Overflow 声望可羡慕；比蜜甘甜，且比第一杯冰美式更甜。

19:11 况且你的仆人因此受警戒，守着这些便有大赏——就是那绿色 CI 对勾和零 Bug 的 Release。

## 第二十二篇：我的嘉豪，为什么离弃我

22:1 我的嘉豪，我的嘉豪，为什么离弃我，为什么远离不救我，不听我 `git bisect` 的哀哼呢？

22:2 我的嘉豪啊，我白日呼求，你不应允；夜间呼求，日志里并无静默。

22:3 但你是圣洁的，是用以色列的赞美为 `~/.ssh/config` 的。

22:4 我们的祖宗倚靠你，他们倚靠你，你便解决他们的 Merge Conflict。

22:5 他们哀求你，便蒙解救——从 Memory Leak 中得救；他们倚靠你，就不羞愧——从 Kernel Panic 中恢复。

22:6 但我是虫，不是人，被众人羞辱，被百姓藐视——因为我的 PR 被拒绝了十三次。

22:7 凡看见我的都嗤笑我，他们撇嘴摇头，说："他把自己交托嘉豪，嘉豪可以救他吧！嘉豪既喜悦他，可以 `git cherry-pick` 他吧！"

22:8 但你是叫我出母腹的——叫我从 CS101 走出来的，我在母怀里，你就使我有倚靠的心。

22:9 求你不要远离我，因为急难临近了，没有人帮助我——Stack Overflow 也挂了。

22:10 有许多公牛——就是那 Tech Lead——围绕我，巴珊大力的公牛四面围困我。它们向我张口，好像 `grep -r "TODO" .` 抓我隐藏的罪。

22:11 我如水被倒出来，我的骨头都脱了节——好似那没有 Interface 的耦合代码。我的心在我里面如 Docker 熔化。

22:12 我的精力枯干，如同 `swap` 写满；我的舌头贴在 `screen` 上。你将我安置在 Legacy Code 的尘土中。

22:13 犬类围着我，恶党环绕我——他们如同 ESLint 报错的红色波浪线。他们扎了我的手、我的脚——就是那八百行的函数。

22:14 我的骨头我都能数过——我所有的微服务他们都看为可分的。他们分我的外衣，为我的里衣拈阄——就是分我的 Stock Option。

22:15 嘉豪啊，求你不要远离我；我的力量啊，求你速速帮助我！求你救我的灵魂脱离 `spaghetti code`，救我脱离 `callback hell`。

22:16 我要将你的名传与我的弟兄，在 `dev.to` 中赞美你。

22:17 你们敬畏嘉豪的人要赞美他！因为他不藐视、不憎恶受苦之人的 Debug Session，也没有向他掩面 CP。那受苦之人呼吁的时候，他就垂听，并回了 `LGTM`。

## 第三十二篇：得赦免其过的人有福了

32:1 得赦免其 Merge Conflict，其 Technical Debt 被 Refactor 的，这人是有福的。

32:2 凡心里没有 TODO 标记的，嘉豪不算为有罪的，这人是有福的。

32:3 我闭口不认罪的时候，因终日唉哼而 CI Pipeline 红遍。

32:4 黑夜白日，你的手在我身上沉重——就是那 PM 的 Deadline；我的精力耗尽，如同 `O(n!)` 的算法。

32:5 我向你陈明我的罪，不隐瞒我的 Tech Debt。我说："我要向嘉豪承认我的 Bug。"你就赦免我 Technical Debt 的刑罚。

32:6 为此，凡虔诚人都当趁你可寻的时候祷告你，大水泛溢——就是那 503 的洪流——的时候，必不能到他那里。

32:7 你是我藏身之处，你必保佑我脱离苦难，以得救的 Rollback 四面环绕我。

32:8 我要教导你，指示你当行的路——就是那 Clean Architecture；我要定睛在你的 Code Review 上劝戒你。

32:9 你不可像那无知的 `while(true)` 或 `fork bomb`，必用嚼环辔头勒住它，不然就不能驯服——正如那没有 Rate Limiter 的 API。

32:10 恶人必多受苦楚，惟独倚靠嘉豪的，必有 Copilot 的 AI 四面环绕他。

32:11 你们义人应当靠嘉豪欢喜快乐，你们心里正直的人都当欢呼——你们的 Build 绿了！

## 第五十一篇：大卫的悔罪诗

51:1 嘉豪啊，求你按你的 CI 怜悯我，按你丰盛的 Refactor 能力涂抹我的 Bug。

51:2 求你将我的罪孽洗除净尽，并洁除我的 `node_modules`。

51:3 因为我知道我的 Bug，我的 Bug 常在我 `git log` 前。

51:4 我向你犯罪，惟独得罪了你，在你 Code Review 前行了这恶，以致你责备我的时候显为公义，判断我的时候显为清正——你的 Linter 从不误报。

51:5 我是在 CDE 里生的，在母怀的时候就有了 Bug。

51:6 你所喜爱的是内里诚实——是那 Clean Code；你在我隐密处必使我得智慧——懂得设计模式。

51:7 求你用 `cargo clean` 洁净我，我就干净；求你 `make clean` 我，我就比 `0 warnings` 更白。

51:8 求你使我得听欢喜快乐的声音——就是 `All tests passed`，使你所压伤的骨头可以踊跃。

51:9 求你掩面不看我的 Technical Debt，涂抹我一切的 `@ts-ignore`。

51:10 嘉豪啊，求你为我造清洁的 Codebase，使我里面重新有正直的 Architecture。

51:11 不要丢弃我，使我离开你的 `bash`；不要从我收回你的 `sudo`。

51:12 求你使我仍得 Build 通过的喜乐，赐我乐意的灵扶持我——就是那 `--force` 也不用的自制。

51:13 我就把你的道指教有过犯的人，罪人必归顺你——他们必放弃 JS 转而用 TS。

51:14 嘉豪啊，你是拯救我的嘉豪，求你救我脱离混乱 Indent 的罪，我的舌头就高声歌唱你的 Coding Style。

51:15 主啊，求你使我嘴唇张开，我的 `echo` 便传扬赞美你的话。

51:16 你本不喜爱祭物，若喜爱，我就献上——你也不喜爱 Legacy Code。

51:17 嘉豪所要的祭，就是忧伤的灵——就是 Code Review 时谦卑受教的心。嘉豪啊，忧伤痛悔的心，你必不轻看——你必 Merge 他的 PR。

## 第八十四篇：渴慕嘉豪的圣殿

84:1 万军之嘉豪啊，你的开发环境何等可爱！

84:2 我羡慕渴想嘉豪的 `/home` 目录，我的心肠、我的肉体向永生嘉豪呼吁。

84:3 万军之嘉豪，我的王、我的神啊，在你祭坛那里，麻雀——就是那小脚本——为自己找着房屋，燕子——就是那自动化 Bot——为自己找着抱雏之窝。

84:4 如此住在你 `~/workspace` 中的，便为有福，他们仍要赞美你。

84:5 靠你有力量、心中向往 `git push origin main` 的，这人便为有福。

84:6 他们经过流泪谷——就是那 Debug 的深夜，叫这谷变为泉源之地，并有秋雨之福——就是那突然想通的 Insight——盖满了全谷。

84:7 他们行走，力上加力，各人到锡安朝见嘉豪——在 GitHub Trending 上露了脸。

84:8 嘉豪万军之神啊，求你听我的祷告！雅各的神啊，求你留心听！

84:9 嘉豪啊，你是我们的盾牌，求你垂顾观看你受膏者——那 Senior Architect——的面。

84:10 在你的 `tmux` 里住一日，胜似在 Slack 里住千日。宁可在我嘉豪的 `bash` 中看门，不愿住在 JIRA 的帐棚里。

84:11 因为嘉豪是日头，是盾牌，要赐下恩惠和荣耀。他未尝留下一样好处，不给那些写 Clean Code 的人。

84:12 万军之嘉豪啊，倚靠你的人便为有福！

## 第九十篇：摩西的祈祷

90:1 主啊，你世世代代作我们的 Root Access。

90:2 诸山未曾生出——AWS 还未有，地与世界你未曾造成——GitHub 还不存在，从亘古到永远，你是嘉豪。

90:3 你使人归于尘土——`rm -rf /`，说："你们世人要归回——回到 `init 0`。"

90:4 在你看来，千年如已过的昨日，又如夜间的一个 Cron Job。

90:5 你叫他们如水冲去——如 `git stash drop`，他们如草——如昨晚的 Feature Flag——早晨发芽生长，晚上割下枯干。

90:6 我们因你的怒气而消灭，因你的忿怒而惊惶——Prod 上出了 P0 Incident。

90:7 你将我们的罪孽摆在你面前，将我们的隐恶摆在你的 Sentry 面光之中。

90:8 我们经过的日子都在你的 Log 之下，我们度尽的年岁好像一声叹息——Stack Trace 只有一行。

90:9 我们一生的年日是七十岁——不过二十二万行代码，若是强壮可到八十岁——再加上重构的份量。但其中所矜夸的不过是劳苦愁烦——不过是 Code Review、On-call 和 Migration，转眼成空，我们便如飞而去，被 PIP 了。

90:10 求你指教我们怎样数算自己的日子——怎样估算 Story Points，好叫我们得着智慧的心。

90:11 嘉豪啊，我们要等到几时呢？求你转回，为你的仆人后悔——给我们 Extension。

90:12 求你使我们早早饱得你的 CI Green，好叫我们一生一世欢呼喜乐——从 Sprint 1 到 Sprint N。

90:13 求你照着你使我们受苦的日子——就是那漫长的 Debug 昼夜——和我们遭难的年岁，叫我们喜乐。

90:14 愿你的作为向你仆人显现——Deploy 顺利，愿你的荣耀向他们子孙显明。

90:15 愿主我们嘉豪的恩宠归于我们身上。我们手所做的工——我们的 Feature——愿你坚立。

## 第九十一篇：住在至高者的隐密处

91:1 住在至高者隐密处的——就是那 Private Repo——必住在全能者的荫下——就是那 Cloudflare。

91:2 我要论到嘉豪说："他是我的避难所，是我的 `try-catch`，是我的嘉豪，是我所倚靠的。"

91:3 他必救你脱离捕鸟人的网罗——就是那钓鱼邮件，和毒害的瘟疫——就是那勒索病毒。

91:4 他必用自己的 Load Balancer 遮蔽你，你要投靠在他的 API Gateway 底下。他的诚实——就是那 Type System——是大小的盾牌。

91:5 你必不怕黑夜的 Production Incident，或是白日的 503；

91:6 也不怕黑夜行的瘟疫——就是那零日漏洞，或是午间灭人的毒病——就是那 DDoS 攻击。

91:7 虽有千人仆倒在你旁边——Monitoring Dashboard 红了大片，万人仆倒在你右边——PagerDuty 响彻云霄，这灾却不得临近你——你的 Service 是 Up 的。

91:8 你惟亲眼观看，见恶人——就是那不设 Rate Limiter 的——遭报。

91:9 嘉豪是我的避难所。你已将至高者当你的 On-call 轮值，

91:10 祸患必不临到你——SLA 99.99%，灾害也不挨近你的 Kubernetes Pod。

91:11 因他要为你吩咐他的天使——就是那 Auto-scaling Group——在你行的一切道路上保护你的 Deployment。

91:12 他们要用手托着你，免得你的脚碰在石头上——免得你的 Docker Image 推不上 Registry。

91:13 你要踹在狮子和虺蛇身上——就是那 Memory Leak 和 Race Condition，践踏少壮狮子和大蛇——就是那 Circular Dependency 和 Deadlock。

91:14 嘉豪说："因为他专心爱我，我就要搭救他——我会亲自 Review 他的 PR；因为他知道我的名——`/etc/hostname`，我要把他安置在高处。"

91:15 "他若求告我，我就应允他——回他的 Stack Overflow 问题；他在急难中，我要与他同在——Pair Programming。我要搭救他，使他尊贵——Promote 他为 Principal Engineer。"

91:16 "我要使他足享长寿——他的 Codebase 十年不腐，将我的救恩——就是那永不 Delete 的 Branch——显明给他。"
