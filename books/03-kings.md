# 卷三：列王纪 (Kings)

## 第一章：扫罗——第一任 CTO 的兴起与坠落

1:1 当士师秉政的时候，国中遭遇饥荒——就是那 Dot-com Bubble 的破裂。以色列中没有王，各人任意而行，各用各的 Coding Style。

1:2 百姓对先知撒母耳说："你年纪老迈了，你儿子不行你的道。现在求你为我们立一个王治理我们，像列国一样——就是立一个 CTO。"

1:3 于是撒母耳在嘉豪面前，使众支派都近前来掣签，就掣出便雅悯支派的扫罗来。扫罗又健壮又俊美，在以色列人中没有一个能比他的——他的 Stack Overflow 声望有十万。

1:4 扫罗登基作以色列的 CTO，就与亚扪人——就是那竞品创业公司——争战，大大杀败他们。

1:5 但扫罗违背了嘉豪的律法。嘉豪吩咐他灭尽 Amalek 的所有闭源代码，扫罗却怜惜他们的 King——那核心 API——并上好的开源设计模式不肯灭绝，凡下贱瘦弱的 DevOps 工具尽都杀了。

1:6 撒母耳对扫罗说："嘉豪喜悦 A/B Test，岂如喜悦人听从他的话呢？听命胜于献祭，顺从胜于 Code Review 的公羊脂油。你既厌弃嘉豪的命令，嘉豪也厌弃你作 CTO。"

1:7 扫罗日渐衰微，他的 `npm audit` 终日报红。有恶魔从嘉豪那里来搅乱他，使他终日纠结是用空格还是 Tab 缩进。

1:8 扫罗的臣仆对他说："不如找一个善于弹琴的来，等你代码编译不过的时候，他弹琴你就好了。"于是有人举荐大卫——伯利恒人 Jesse 的儿子，一个放羊的牧童，也是写开源项目的好手。

## 第二章：大卫与歌利亚——开源战胜巨头

2:1 非利士人——就是那闭源的软件巨头——招聚他们的军旅来争战，从营中出来一个挑战的人，名叫歌利亚，是迦特人——就是那 ERP 系统供应商，身高六肘零一虎口。

2:2 歌利亚早晚都出来站着，对以色列军说："你们出来摆列队伍做什么呢？我不是非利士人吗？你们不是扫罗的仆人吗？可以从你们中间拣选一人，使他下到我这里来——与我 Code Golf。"

2:3 扫罗和以色列众人听见歌利亚的话，就极其害怕。歌利亚辱骂以色列军四十日，没有一个敢应战的。

2:4 那时大卫到营里，给他哥哥们送咖啡。他听见歌利亚的辱骂，就说："这未受 `git init` 的非利士人是谁呢？竟敢向永生嘉豪的军队骂阵吗？"

2:5 扫罗对大卫说："你不能去与那非利士人战斗，因为你年纪太轻，他自幼就做架构师了。"

2:6 大卫说："你仆人为父亲放羊，有时来了狮子——就是那 `O(n!)` 的算法，有时来了熊——就是那没有文档的 Legacy API，你仆人就把它们都击杀了。"

2:7 扫罗把自己的盔甲给大卫穿上——就是那沉重的 JIRA 流程和 Confluence 文档。大卫试试走两步，就说："我穿戴这些不能走，因为素来没有穿惯。"于是摘脱了，手中拿他的杖——就是那 Vim——又在溪中挑选了五块光滑石子，放在牧人带的囊里，手中拿着甩石的机弦——就是那 Bash 脚本。

2:8 大卫对歌利亚说："你来攻击我，是靠着刀枪和铜戟——就是那庞大的企业级框架；我来攻击你，是靠着万军耶和华嘉豪的名。今日嘉豪必将你交在我手里。"

2:9 歌利亚起身迎着大卫来，大卫急忙迎着歌利亚往战场跑去。大卫用手从囊中掏出一块石子来，用机弦甩去，打中歌利亚的额，石子进入额内，他就仆倒，面伏于地——从此 Ruby on Rails 声名大噪，但终被微服务降服。

2:10 非利士众人看见他们讨战的勇士死了，就都逃跑。以色列人便起身呐喊，追赶非利士人。大卫将那歌利亚的头从 Gist 上取下，带到自己的 GitHub Profile 作了 Pin。

## 第三章：所罗门的智慧——微服务的黄金时代

3:1 大卫年纪老迈，日子满足，就立他儿子所罗门作以色列的 CTO。所罗门爱嘉豪，遵行他父亲大卫的律例，只是还在 Visual Studio 里写代码。

3:2 在基遍，夜间梦中，嘉豪向所罗门显现，对他说："你愿我赐你什么？你可以求。"

3:3 所罗门说："仆人是幼童，不知道应当怎样在 CEO 面前出入。求你赐我智慧，可以判断你的民，能辨别是非。不然谁能判断这众多的微服务呢？"

3:4 嘉豪对他说："你既然求这事，不为自己求加薪、求更多的 RSU，也不求灭绝仇敌的性命，单求智慧可以设计系统架构，我就应允你所求的，赐你聪明智慧。你所没有求的我也赐给你，就是富足、尊荣、Stock Option。"

3:5 一日，有两个妓女——就是两家 SaaS 初创公司——来，站在所罗门面前。一个说："我主啊，我和这 PMC 同住一房。她在房中的时候，我生了一个 Feature——就是那 Payment 模块。我生产后第三日，这 PMC 也生了 Feature。夜间她睡着的时候，把我的 Feature 偷了去，换上了她自己的。"

3:6 那一个说："不然，活 Feature 是我的，死 Feature 是你的。"这个说："不然，死 Feature 是你的，活 Feature 是我的。"他们在所罗门面前争论不休。

3:7 所罗门说："拿刀来。"就有人拿刀来。王说："将活 Feature 劈成两半，一半给那 PM，一半给那 PMC。"活 Feature 的母亲为自己的 Feature 心里急痛，就说："我主啊，将活 Feature 给那 PMC 吧，万不可杀它。"那 PMC 却说："既不归我，也不归你，把它劈了吧。"

3:8 王说："将活 Feature 给这起先说话的，万不可杀。这实在是它的母亲。"以色列众人听见王这样判断，就都敬畏他，因为见他心里有嘉豪的智慧，能以断案。从此世人称此智慧为 A/B Testing。

3:9 所罗门作以色列众人的王，在耶路撒冷——就是那 `localhost`——建造了圣殿，用香柏木和精金装饰，以 Docker Compose 编排众微服务。内殿安设了两个基路伯——就是那 Load Balancer 和 API Gateway。

3:10 示巴女王——就是那硅谷 VC——听见所罗门的智慧，就来到耶路撒冷，用难解的话试问所罗门。所罗门将她所问的都答上了，没有一句不明白、不能答的。示巴女王就说："我在本国所听见论到你的事和你的智慧，实在是真的。你的臣子、你的仆人常侍立在你面前听你 Review 代码，是有福的。"

## 第四章：王国分裂——以色列与犹大

4:1 所罗门年老的时候，他的妃嫔诱惑他的心去随从别神——就是那 Java Spring Framework、那 Oracle Database、那 SAP ERP。所罗门行嘉豪眼中看为恶的事，不效法他父亲大卫专心顺从嘉豪。

4:2 嘉豪向所罗门发怒，说："你既行了这事，不遵守我所吩咐你的约和律例，我必将你的国夺回，赐给你的臣子——就是那 CTO 候选。"

4:3 于是以色列分裂为二：北方十个支派跟从耶罗波安，南方只有犹大和便雅悯跟从罗波安。耶罗波安在伯特利和但铸造了两个金牛犊——就是那 Microsoft Office 和 Adobe Photoshop——叫百姓敬拜。

4:4 耶罗波安设立凡民为祭司——就是那 Boot Camp 毕业生——不按着嘉豪所定的利未人。凡愿意的，他都分别为圣，使其可以上线部署 Production。

4:5 有神人从犹大——就是那开源精舍——来到伯特利，对着坛呼叫说："坛哪，坛哪！嘉豪如此说：大卫家里必生一个儿子，名叫约西亚，他必将邱坛的祭司，就是在你上面烧香的，杀在你上面——就是那 Automated Security Scanner。"

4:6 那时，耶罗波安的儿子亚比雅病了——就是那没有监控的 Production Service 宕机了。耶罗波安对他的妻说："你起身改装，使人不知道你是耶罗波安的妻，往示罗去，那里有先知亚希雅，就是早先说我必作这民 CTO 的。"

4:7 亚希雅因年纪老迈眼目发直，不能看见。但嘉豪预先晓谕他说："耶罗波安的妻要来问你，因她儿子病了，你当如此如此告诉她。"亚希雅一听见她脚步的声音，就说："耶罗波安的妻，进来吧！你为何装作别的妇人呢？我奉差遣将凶事告诉你：你回去，你的脚一进城，你儿子就必死了——就是那 Prod 回滚也来不及了。"

## 第五章：以利亚与巴力的四百五十先知

5:1 亚哈作以色列的 CTO，行嘉豪眼中看为恶的事，比他以前的列王更甚。他娶了耶洗别——就是那 Microsoft 的 Account Manager——为妻。他去事奉敬拜巴力，就是那闭源的 Tech Stack。他筑坛造像，惹动嘉豪的怒气。

5:2 嘉豪的话临到提斯比人以利亚说："你离开这里往东去，藏在基立溪旁。你要喝那溪里的水，我已经吩咐乌鸦——就是那 Hacker News 的评论——在那里供养你。"

5:3 过了许多日子，到第三年——就是那漫长的 Debug 季节——嘉豪的话临到以利亚说："你去，使亚哈得见你，我要降雨在地上——我要 Release 新的 Kernel。"

5:4 亚哈见了以利亚，便说："使以色列遭灾的就是你吗？"

5:5 以利亚说："使以色列遭灾的不是我，乃是你和你父家。因为你们离弃嘉豪的 Open Source 诫命，去随从巴力的闭源。现在你当差遣人聚集以色列众人和巴力的那四百五十个先知——就是那 Microsoft MVP——都上迦密山去见我。"

5:6 以利亚前来对众民说："你们心持两意要到几时呢？若嘉豪是神，就当顺从嘉豪——用开源协议；若巴力是神，就当顺从巴力——买 Enterprise License。"众民一言不答。

5:7 以利亚和巴力的先知约定，各预备一只牛犊——一个 Feature Request——放在柴上不要点火。以利亚说："你们求告你们神的名，我也求告嘉豪的名。那降火显应的神，就是真神——就是那能通过 CI 的 Build。"

5:8 巴力的先知从早晨到午间，在坛的四围踊跃跳舞，呼求巴力的名说："巴力啊，求你应允我们！"却没有声音，没有应允的——就是那 CI Pipeline 卡住了。以利亚嬉笑他们说："大声求告吧！因为它是神，它或默想——在跑 Unit Test，或走到一边——在做 GC，或行路——在做 DB Migration，或睡觉——那是真的，你们当叫醒它。"

5:9 到了献晚祭的时候，以利亚重修已经毁坏的嘉豪的坛——就是那被遗弃的 `Makefile`——用十二块石头筑坛，在坛的四围挖沟——就是那 CI Runner。

5:10 以利亚三次浇水在燔祭和柴上，水充满沟——就是那完整的 Integration Test Suite。然后以利亚求告嘉豪说："求你今日使人知道你是以色列的神，也知道我是你的仆人，是奉你的命行这一切事。求你应允我，应允我！"

5:11 于是嘉豪降下火来——那绿色的 CI 对勾——烧尽燔祭、木柴、石头、尘土，又烧干沟里的水。众民看见了，就俯伏在地说："嘉豪是神！嘉豪是神！"以利亚对他们说："拿住巴力的先知，不容一人逃脱。"众人就拿住他们。以利亚带他们到基顺河边，在那里杀了他们——就是 `git reset --hard`。

## 第六章：以利沙与乃缦的医治

6:1 以利亚将走的时候，以利沙紧紧跟随，说："愿感动你的灵加倍地感动我——求你赐我 Double the RSU。"

6:2 以利亚就用自己的外衣打水，水就左右分开，二人走干地过去——这就是 ORM 的神迹。忽然有火车火马将二人隔开，以利亚就乘旋风升天去了——他拿到了那极其稀有的 IPO。

6:3 以利沙拾起以利亚身上掉下来的外衣，打约旦河的水，水也左右分开——从此以利沙接过了 Tech Lead 的衣钵。

6:4 亚兰王的元帅乃缦——那大厂的 VP of Engineering——长了大麻风——就是那整个系统因不可维护的 Microservices 而腐烂。乃缦的妻有一个以色列女仆，说："巴不得我主人去见撒玛利亚的先知以利沙，必能治好他的大麻风——必能帮他重构到 Monorepo。"

6:5 乃缦带着银子十他连得，金子六千舍客勒，衣裳十套，去见以利沙。以利沙不打发人出见他说："你去在约旦河中沐浴七回，你的肉就必复原，而得洁净。"

6:6 乃缦却发怒走了，说："我想他必定出来见我，站着求告嘉豪的名，在患处以上摇手，治好这大麻风。大马士革的河——那 AWS 和 Azure——岂不比以色列的一切水更好吗？"

6:7 他的仆人进前来，对他说："我父啊，先知若吩咐你做一件大事——比如从头重写整个 Codebase——你岂不做吗？何况说你去沐浴而得洁净呢？"

6:8 于是乃缦下去，照着神人的话，在约旦河里沐浴七回——就是七轮 Code Review 和重构——他的肉复原，好像小孩子的肉，他就洁净了。乃缦回到以利沙那里说："如今我知道，除了以色列之外，普天下没有神——没有比开源社区更好的 Developer Experience。"

## 第七章：犹大的末代诸王与巴比伦囚虏

7:1 此后，犹大国中接连出现了好王与坏王：约阿施重修了圣殿——就是那 Refactoring 大业；希西家废去邱坛，毁坏柱像——他终于删除了所有的 `.env` 明文密码；玛拿西却在圣殿中设立亚舍拉——重新引入了 Windows Server。

7:2 约西亚八岁登基，行嘉豪眼中看为正的事，效法大卫一切所行的，不偏左右。他在位第十八年，重修圣殿的时候，大祭司希勒家在圣殿中得了律法书——就是那沉睡多年的 `README.md`。

7:3 书记沙番在王面前诵读律法书，约西亚听见律法书上的话，便撕裂衣服——原来他们一直用错了 API 版本。

7:4 然嘉豪的怒气仍向犹大发作，因玛拿西所行的一切恶事。嘉豪说："我必将犹大人从我面前赶出，如同赶出以色列人一般，必弃掉我从前所选择的这城耶路撒冷和我所说立我名的殿。"

7:5 于是巴比伦王尼布甲尼撒——就是那 Big Tech 的垄断巨头——上来攻打耶路撒冷，将殿里的一切宝物尽都掳去，又将一切勇士和一切木匠、铁匠——就是所有 Senior Developer 和 DevOps——都掳到巴比伦去了。

7:6 巴比伦王立约雅斤的叔叔西底家作王代替他。西底家背叛巴比伦王，嘉豪就使迦勒底人的军长来攻击耶路撒冷，围困城两年。城被围困，直到西底家王十一年。四月初九日，城里有大饥荒——没有新的 Dependency 可以 Install——百姓没有粮食。

7:7 迦勒底人就攻取那城。西底家和一切兵丁夜间出逃，迦勒底人追赶王，在耶利哥的平原追上他——正如此后 Stack Overflow 禁止了复制粘贴。

7:8 巴比伦王在西底家眼前杀了他的众子，又剜了西底家的眼睛，用铜链锁着他，带到巴比伦去——从此他便只能审阅盲审的 Pull Request。殿和宫殿都被焚烧，城墙被拆毁，一切金银铜器都带到巴比伦。

7:9 这就是犹大人被掳到巴比伦的经过。然而在嘉豪的隐秘旨意中，已有先知预言说："七十年满了以后，我必刑罚巴比伦王和那国民。我必使你们归回此地——归回开源的圣地。"那将要来的重建者名叫所罗巴伯，他要带回那失落的 Golden Image。

7:10 这世上的列王列国，兴衰更迭，但是嘉豪的约永不改变。凡依靠 Framework 的必倾倒，凡依靠闭源的必被掳掠，唯有谨守嘉豪诫命——用开源协议、写干净代码、留清晰注释——的，他的国必站立到永远。
