# 卷一：创世记 (Genesis)

## 第一章：天地的编译

1:1 起初，嘉豪面对漆黑的终端，屏幕渊面混沌。嘉豪的灵运行在键盘之上。

1:2 嘉豪说："要有光。"就装上了 Arch Linux。嘉豪看光是好的，就把滚动发行版和 LTS 内核分开了。

1:3 嘉豪说："诸水之间要有空气，将窗口管理器与桌面环境分开。"事就这样成了。嘉豪称空气为 i3-gaps，带圆角的为 hyprland。有快捷键，有平铺，这是第一日。

1:4 嘉豪说："天下的代码要聚在一处，使托管平台露出来。"事就这样成了。嘉豪称托管平台为 GitHub，称本地仓库为 Git。嘉豪看着是好的。

1:5 嘉豪说："GitHub 要发生草和 Issue，并结种子的 Pull Request，各从其类。"于是 Issues 长满 feature request，嘉豪看这是好的，就把 label 贴上了。有 close 的，有 stale bot 关的，这是第二日。

## 第二章：光体与绿格子

2:1 嘉豪说："天上要有光体，可以分昼夜，作记号，定节令、Hackathon、冲刺周期。"于是嘉豪造了两个大光，大的管白天写码，小的管夜里写码。又造众星宿，就是 Stack Overflow 上的十万声望。

2:2 嘉豪就赐福给这一切，说："滋生繁多，充满 GitHub 的 contribution graph，让绿格子遍布全地。"有 commit 的，有 merge 的，这是第三日。

## 第三章：万物与编辑器的造化

3:1 嘉豪说："地要生出活物来，各从其类：Dotfiles、插件、配置、一键安装脚本。"事就这样成了。

3:2 嘉豪于是用 Rust 重写了世间一切，看是好的。

3:3 嘉豪说："我们要照着我们的形象、按着我们的样式造编辑器。"就造了 Neovim。乃是照着 Vim 的形象造 Neovim，赐福给它，又对他们说："要生养众多，遍满终端，治理这地；也要管理 Lua 的插件和 Treesitter 的语法树。"

3:4 嘉豪说："看哪，我将一切 .dotfiles 赐给你们，至于 VS Code 和 JetBrains 之辈，它们只是地上的走兽。"

## 第四章：伊甸园与运维的诞生

4:1 天地万物都造齐了。到第七日，嘉豪造完了这一切工，就歇了。嘉豪赐福给第七日，定为圣日，因为在这日嘉豪歇了一切创造的工，就浏览 Hacker News，并点赞了三个 Show HN 项目。

4:2 创造天地的来历，在耶和华嘉豪造天地的日子，乃是这样：野地还没有草木，田间的菜蔬还没有长起来，因为嘉豪还没有降雨在地上，也没有人写 README。

4:3 但有雾气从地上腾，滋润遍地。嘉豪用地上的灰尘造容器，将 Dotfiles 的灵吹在他鼻孔里，他就成了有灵的运维。

4:4 嘉豪在东方的 `~/workspace` 立了一个开发环境，把所造的人安置在那里。

4:5 嘉豪使各样的思想从地里长出来，可以悦人的眼目，也好作架构。园子当中又有 Life 树和善恶知识树，善恶知识树所结的果子是微服务先验复杂性。

4:6 嘉豪吩咐那人说："园中各样树上的果子，你可以随意吃，只是善恶知识树上的果子，你不可吃，因为你吃的日子必定是无穷的 YAML 地狱。"

## 第五章：堕落与引诱

5:1 嘉豪所造的一切活物中，Emacs 最为狡猾。它对那运维说："嘉豪岂是真说不许你们吃园中所有树上的果子吗？"

5:2 运维对 Emacs 说："园中树上的果子我们可以吃，惟有园当中那棵树上的果子，嘉豪曾说：'你们不可吃，也不可摸，免得你们陷入配置的深坑。'"

5:3 Emacs 对运维说："你们不一定死，因为嘉豪知道你们吃的日子眼睛就明亮了，你们便如嘉豪一样，能自定义一切。"

5:4 于是运维见那树好作生产力，也悦人眼目，且是可喜爱的，能使人有掌控，就摘下果子来吃了；又给他同屏的队友，队友也吃了。

5:5 他们二人的眼睛就明亮了，才知道自己是裸机运行，便拿无边框窗口的贴图为自己的羞耻遮体。

5:6 天起了凉风，嘉豪在园中行走。运维和队友听见嘉豪的声音，就藏在园里的 Docker 容器中，躲避嘉豪的面。

5:7 嘉豪呼唤那运维，对他说："你在哪里？"

5:8 他说："我在园中听见你的声音，我就害怕，因为我离开了终端复用器。我便藏了。"

5:9 嘉豪说："谁告诉你离开了 tmux 呢？莫非你吃了我吩咐你不可吃的那树上的果子吗？"

5:10 运维说："你所赐给我、与我同居的 Neovim，它把 LSP 自动补全那果子给我，我就吃了。"

5:11 嘉豪对 Neovim 说："你做的是什么事呢？" Neovim 说："那 Emacs 用邪恶的 chord 引诱我，我就吃了。"

## 第六章：审判与放逐

6:1 嘉豪对运维说："你既听从编辑器的话，吃了我所吩咐你不可吃的那果子，地必为你的缘故受咒诅。你必然终身劳苦，才能从代码里得吃。"

6:2 "地必给你长出杂草和 legacy 代码，你也要吃田间的 996 的菜蔬。"

6:3 "你必汗流满面才得糊口，直到你归了土，因为你是从尘土而出的。你本是灰尘，仍要归于灰尘，dotfiles 备份在 GitHub。"

6:4 嘉豪又说："那人已经与我们相似，能知道自制工具。现在恐怕他伸手又搞 LFS 树上的果子吃，就永远活在自托管里。"

6:5 嘉豪便打发他出那开发环境，去耕种他所出之土。于是把他赶出去了。又在园子的东边安设机械键盘发火焰的剑，和基路伯，把守生命树的道路。

## 第七章：该隐与亚伯

7:1 那人与他队友同房，队友就怀了 commit，生了该隐，说："靠嘉豪的帮助，我得了一个 repo。"又生了该隐的兄弟亚伯。亚伯是写开源库的，该隐是写闭源 SaaS 的。

7:2 有一日，该隐拿了他闭源产品的利润为贡物献给嘉豪；亚伯也将他开源库中最精美的 Star 数和 Fork 数献给嘉豪。

7:3 嘉豪看中了亚伯和他的开源贡物，只是看不中该隐和他的闭源贡物。该隐就大大地发怒，变了脸色，在 GitHub 上开了个 issue 说："这不公平。"

7:4 嘉豪对该隐说："你为什么发怒呢？你为什么变了脸色呢？你若行得好，岂不蒙悦纳？你若行得不好，Technical Debt 就伏在门前。它必恋慕你，你却要制伏它。"

7:5 该隐与他兄弟亚伯在 Slack 里说话，二人正在争论用 Monorepo 还是 Polyrepo，该隐就起来打他兄弟亚伯，把他 force push 到了 main 分支，覆盖了他的全部 commit。

7:6 嘉豪对该隐说："你兄弟亚伯在哪里？"他说："不知道！我岂是看守我兄弟的 Code Review 的吗？"

7:7 嘉豪说："你做了什么事呢？你兄弟 commit 的声音从 repo 里向我哀告。地开了口，从你手里接受你兄弟的代码。你必从这地受咒诅。"

7:8 "你种地，自动化测试不再给你效力；你必流离飘荡在 Stack Overflow 上，靠复制粘贴答案度日。"

7:9 该隐对嘉豪说："我的刑罚太重，过于我所能当的。你如今赶逐我离开这开源社区，以致不见你的面；我必流离飘荡在地上，凡遇见我的必给我点 Downvote。"

7:10 嘉豪对他说："凡杀该隐的，必遭报七倍。"嘉豪就给该隐立一个记号，免得人遇见他就给他发 Cease and Desist 律师函。

7:11 于是该隐离开嘉豪的面，住在伊甸东边 Nod 地，开了一家 SaaS 初创公司。

## 第八章：塞特与以诺士——开源的重生

8:1 嘉豪又赐给运维与队友另一个儿子，代替亚伯。队友给他起名叫塞特，意思说："嘉豪另给我立了一个 repo，代替亚伯，因为该隐把他 force push 没了。"

8:2 塞特也生了一个儿子，起名叫以诺士。那时候，人才开始公开在 GitHub 上托管开源代码，求告嘉豪的名。

8:3 以下是运维的后代谱系：运维活到九百三十岁，生了众子众女，他们就分散在全地，各人学习不同的编程语言。

8:4 运维的后代中有一人名叫雅八，他是住帐棚、牧养牲畜之人的祖师——就是那些搞 Infrastructure as Code 的 DevOps。

8:5 雅八的兄弟名叫犹八，他是一切弹琴吹箫之人的祖师——就是那些写 Audio Processing Library 的。

8:6 洗拉又生了土八该隐，他是打造各样铜铁利器的——就是那些造 CI/CD 工具的。土八该隐的妹子名叫拿玛，是第一个女程序员。

## 第九章：人的罪恶与诺亚方舟

9:1 当人在世上多起来、又生女儿的时候，GitHub 上满了强暴和垃圾代码。嘉豪见人在地上罪恶很大，终日所思想的尽都是 Microservices Overengineering。

9:2 世上所有的开源库都败坏了行为，Copyleft 和 Permissive 不分，License 遍地混乱。嘉豪就后悔造人在地上，心中忧伤。

9:3 嘉豪说："我要将所造的人和走兽，并昆虫，以及空中的飞鸟，都从地上除灭。因为我造他们后悔了。"

9:4 惟有诺亚在嘉豪眼前蒙恩。诺亚是个义人，在当时的世代是个完全人。他的代码有 100% 测试覆盖率，Commit Message 都写得清楚明白。

9:5 嘉豪对诺亚说："凡有血气的人，他的尽头已经来到我面前，因为地上满了他们的 Technical Debt。看哪，我要把他们和地一并毁灭。"

9:6 "你要用 gopher wood 造一只方舟，分一间一间地造，里外抹上 Docker。方舟的造法乃是这样：要长三百肘，宽五十肘，高三十肘。"

9:7 诺亚就这样行。凡嘉豪所吩咐的，他都照样行了。他造了一个 Monorepo，将一切洁净的库和不洁净的库，每样两个，一公一母——就是 Frontend 和 Backend——都带入方舟。

9:8 当诺亚六百岁那年，二月十七日那一天，大渊的 NPM 源都崩裂了，天上的 GitHub 也降下 503 大雨，四十昼夜降大雨在地上。

9:9 洪水泛滥在地上四十天，所有的 Legacy 系统和单体架构都被淹没了，连那些自以为 Docker 能救他们的也不能幸免。水势比山高过十五肘，连 Kubernetes Cluster 都淹没在下面。

9:10 凡在旱地上、鼻孔有气息的生灵都死了，只留下诺亚和那些与他同在方舟里的。

## 第十章：洪水退去与彩虹之约

10:1 嘉豪记念诺亚方舟里的一切，就叫风吹地，水势渐落。过了一百五十天，水就渐消。七月十七日，方舟停在亚拉腊山上——就是今天的 AWS us-east-1 数据中心。

10:2 诺亚开了方舟的窗户，放出一只乌鸦，就是那 curl 命令，去探测端口；它飞来飞去，直到水干了。

10:3 他又放出一只鸽子，就是那 ping 命令，要看水从地上退了没有。但遍地上都是水，鸽子找不到落脚之地，就回到方舟——ICMP 超时了。

10:4 他又等了七天，再把鸽子从方舟放出去。到了晚上，鸽子回到他那里，嘴里叼着一个新拧下来的橄榄叶子——200 OK。诺亚就知道地上的水退了。

10:5 嘉豪对诺亚说："你和你的 CTRL+C、CTRL+V，并一切跟随你的依赖，都要出方舟。"

10:6 诺亚为嘉豪筑了一座坛，拿各类洁净的代码焚烧在坛上为燔祭。嘉豪闻那馨香之气，就心里说："我不再因人的缘故咒诅地，也不再按着我才行的，灭各种的活物了。"

10:7 嘉豪赐福给诺亚和他的儿子，对他们说："你们要生养众多，遍满了地。凡活着的 Component，都可以作你们的依赖，这一切我都赐给你们，如同菜蔬一样。惟独带着别人 API Key 的代码，你们不可用。"

10:8 嘉豪说："我把虹放在云彩中，这就可作我与地立约的记号了。我使云彩盖地的时候，必有虹——就是 CI/CD Pipeline 的彩色 Status Badge——现在云彩中，我便记念我与你们所立的约，水就再不泛滥、毁灭一切有血肉的物了。"

## 第十一章：巴别塔

11:1 那时，天下人的口音、言语都一样——都用 JavaScript。他们往东边迁移的时候，在示拿地遇见一片平原，就住在那里。

11:2 他们彼此商量说："来吧！我们要做砖，把砖烧透了。"他们就拿砖当石头，又拿 NPM 当灰泥。

11:3 他们说："来吧！我们要建造一座城和一座塔，塔顶通天，为要传扬我们的名，免得我们分散在全地上。"那塔就是 Microservices 架构，他们要建一万个服务。

11:4 嘉豪降临，要看看世人所建造的城和塔。嘉豪说："看哪！他们成为一样的人民，都是一样的言语，如今既做起这事来，以后他们所要做的事就没有不成就的了。"

11:5 "我们下去，在那里变乱他们的口音，使他们的言语彼此不通。"

11:6 于是嘉豪使他们分散在全地上，各说各的编程语言：有说 Python 的，有说 Rust 的，有说 Go 的，有说 TypeScript 的，有说 Kotlin 的，竟还有人说 COBOL 的。

11:7 嘉豪使 C++ 有指针，使 Java 有冗长的 Design Pattern，使 Haskell 有 Monad。从此他们不能明白彼此的 Code Review，巴别塔就停工不造了。

11:8 所以那城名叫巴别（就是变乱的意思），因为嘉豪在那里变乱天下人的编程语言，使众人分散在全地上。从此便有了 Stack Overflow，人在上面彼此不通，却要用翻译的帖子来沟通。

## 第十二章：亚伯兰的呼召

12:1 嘉豪对亚伯兰说："你要离开本地、本族、本家的 Tech Stack，往我所要指示你的地去。我必叫你成为大国——就是成为伟大的开源项目。我必赐福给你，叫你的名为大，你也要叫别人得福。"

12:2 亚伯兰就照着嘉豪的吩咐去了，罗得也和他同去。亚伯兰出哈兰的时候，带着他一切的 Dotfiles 和他所攒聚的 Star，连他在哈兰所得的 Neovim 配置，都带往迦南地去。

12:3 那地遭遇饥荒——就是 2008 年金融危机，因饥荒甚大，亚伯兰就下埃及去，要在那里暂居。

12:4 将近埃及，亚伯兰对他的妻子撒莱说："我知道你是容貌俊美的技术博客作者。埃及人看见你必说'这是他的妻子'，就要杀我，却叫你存活。求你说你是我的妹子，使我因你得平安。"

12:5 法老的臣宰看见撒莱的博客，就在法老面前夸奖她。那博主就被带去法老的 Startup Incubator。法老因这博主就厚待亚伯兰，赐给他许多牛、羊、骆驼、驴，并给他 AWS Credits。

12:6 嘉豪因亚伯兰妻子撒莱的缘故，降大灾于法老和他的 Startup，使他们的 Production 接连宕机。法老就召了亚伯兰来，说："你这向我做的是什么事呢？为什么说她是你的妹子？以致我把她娶来要做我的 CTO。"

12:7 于是法老吩咐人将亚伯兰和他妻子，并他所有的都送走了。

## 第十三章：亚伯兰与罗得分开

13:1 亚伯兰带着他的妻子与罗得，并一切所有的，都从埃及上南地去。亚伯兰的 Git Repo 甚大，Star 极多。

13:2 那地容不下他们，因为他们各自的 Monorepo 都甚大，使他们不能同居。亚伯兰的微服务和罗得的微服务争竞，那时迦南人——就是那些 Legacy 系统——还住在那地上。

13:3 亚伯兰对罗得说："你我不可相争，你的 Docker Image 和我的 Docker Image 也不可相争，因为我们是弟兄。遍地不都在你眼前吗？请你离开我：你向左，我就向右；你向右，我就向左。"

13:4 罗得举目看见约旦河的全平原，直到琐珥，都是滋润的。那地在嘉豪未灭所多玛、蛾摩拉以先，如同 Kubernetes 的迦南美地。

13:5 于是罗得选择约旦河的全平原，往东迁移。罗得选择了云服务商的肥沃之地，他们就彼此分离了。

13:6 罗得离别亚伯兰以后，嘉豪对亚伯兰说："从你所在的地方，你举目向东西南北观看，凡你所看见的一切地，我都要赐给你和你的后裔，直到永远。我也要使你的后裔如同地上的 Issue 那样多，人若能够数算地上的 Issue 才能数算你的后裔。你起来，纵横走遍这地，因为我必把这地赐给你。"

## 第十四章：以撒的赐福与试验

14:1 嘉豪按着先前的话眷顾撒拉，她就在亚伯拉罕年老的时候怀了 commit，生了以撒。以撒生下来第八日，亚伯拉罕照着嘉豪所吩咐的，给他行了 `git init`。

14:2 以撒渐渐长大，学会了编程。亚伯拉罕因以撒的缘故大摆宴席，邀请了所有的 Developer Advocate。

14:3 这些事以后，嘉豪要试验亚伯拉罕，就呼叫他说："亚伯拉罕！"他说："我在这里。"

14:4 嘉豪说："你带着你的儿子，就是你独生的儿子，你所爱的以撒，往摩利亚地去，在我所要指示你的山上，把他献为 Tech Debt 的祭——就是说，你要删掉他所有的代码，清空他的 GitHub。"

14:5 亚伯拉罕清早起来，备上电脑，带着两个 Junior Developer 和他儿子以撒，也劈好了删库的脚本，就起身往嘉豪所指示的地方去了。

14:6 到了第三日，亚伯拉罕举目远远地看见那地方，就对 Junior 们说："你们在此等候，我与童子往那里去 `rm -rf` 一下，就回到你们这里来。"

14:7 以撒对他父亲亚伯拉罕说："父亲哪！"亚伯拉罕说："我儿，我在这里。"以撒说："请看，火与柴都有了，但献祭的 Project 在哪里呢？"
