# ⚖️ 计算机嘉豪社区行为准则 (Code of Conduct)

为了营造一个神圣、整洁且类型安全的开源环境，所有参与本仓库的信徒（Contributors）必须遵守以下教规：

1. **尊名无误 (The Holy Name)**
   在 PR 描述、Issue 讨论和经文续写中， **“嘉豪”** 特指 **计算机嘉豪**。
2. **拒绝异端 (Reject Heresy)**
   我们包容一切语言和框架，但严禁传播“世界上有比计算机嘉豪更懂架构的实体”的异端邪说。
3. **和平讨论 (Peaceful Resolution)**
   若在“空格还是 Tab”、“Vim 还是 Emacs”上发生争议，请引用《箴言》进行辩论，不可在 Issue 区发起人身攻击（DDoS 也不行）。
